# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-09

### Added
- Initial stable release
- Support for 200+ templates from @antv/infographic
- Support for 200+ official themes
- PNG export functionality (right-click)
- SVG export functionality (right-click)
- Copy to clipboard functionality (right-click)
- Copy source code functionality (right-click)
- Editable mode for inline editing
- Automatic dark/light theme detection
- Responsive design for mobile and desktop
- BRAT plugin installation support
- Comprehensive documentation and examples

### Templates Included
- List templates: horizontal-arrow, grid-badge-card, grid-compact-card, done-list
- Sequence templates: simple, timeline, snake-steps, ascending-steps, zigzag-steps-underline-text
- Hierarchy templates: tree-curved-line, mindmap-curved-line
- Chart templates: pie, bar, column
- Compare templates: swot, binary-horizontal
- Quadrant templates: quarter-simple-card

### Features
- Powered by official @antv/infographic engine
- Right-click context menu for export operations
- High-resolution PNG export (2x scale)
- SVG vector export with embedded resources
- Theme selection from 200+ official themes
- Inline editing mode (toggle in settings)
- Markdown code block processor (`infographic`)
- Error handling and user-friendly messages
- Comprehensive README with installation guides

### Installation Methods
- BRAT plugin (recommended)
- Manual download from GitHub Releases
- Clone from source with build instructions

### Documentation
- Complete README with syntax reference
- Installation guides for all methods
- Template examples and descriptions
- Theme system documentation
- Export functionality documentation
- Development setup instructions

### Technical Details
- Built with TypeScript
- Bundled with esbuild
- Managed with Bun package manager
- Uses Obsidian Plugin API
- Integrates with @antv/infographic v0.2.13
- Compatible with Obsidian 0.15.0+
